给定一个二叉搜索树（Binary Search Tree），把它转换成为累加树（Greater Tree)，使得每个节点的值是原来的节点值加上所有大于它的节点值之和。

例如：

输入: 二叉搜索树:
              5
            /   \
           2     13

输出: 转换为累加树:
             18
            /   \
          20     13

二叉搜索树：
性质： 左子树 < 根 < 右子树

那么存在累加树：

右孩子 
根的值 = 根的值 + 右子树的值
左 = 左 + 根 + 右

```javascript
var convertBST = function (root) {
    var add = 0
    function dfs(root) {
        if (null == root) return root
        dfs(root.right)
        root.val += add
        add = root.val
        dfs(root.left)
        return root
    }
    return dfs(root)
};
```
