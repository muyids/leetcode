

#### [507. 完美数](https://leetcode-cn.com/problems/perfect-number/)



**解题思路**



需要枚举所有正因子







**朴素解法**



```java
class Solution {
    public boolean checkPerfectNumber(int n) {
        int s = 0;
        for (int i =1; i<n; i++) {
            if (n % i == 0) s+= i;
        }
        return s == n;
    }
}
```





**优化解法**

```java
class Solution {
    public boolean checkPerfectNumber(int n) {
        int s = 0;
        if (n!=1) s+=1;
        int i = 2;
        for (; i * i < n; i++) {
            if (n % i == 0) {
                s+=i;
                s+=n/i;
            }
        }
        if (i*i==n) s+=i;
        return s == n;
    }
}
```
