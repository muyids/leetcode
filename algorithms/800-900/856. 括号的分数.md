
给定一个平衡括号字符串 S，按下述规则计算该字符串的分数：

() 得 1 分。
AB 得 A + B 分，其中 A 和 B 是平衡括号字符串。
(A) 得 2 * A 分，其中 A 是平衡括号字符串。
 

示例 1：

输入： "()"
输出： 1
示例 2：

输入： "(())"
输出： 2
示例 3：

输入： "()()"
输出： 2
示例 4：

输入： "(()(()))"
输出： 6
 

提示：

S 是平衡括号字符串，且只含有 ( 和 ) 。
2 <= S.length <= 50

```javascript
var scoreOfParentheses = function(S) {
    let stack = []
    for (let i = 0; i< S.length; i++){
        if (S[i] == '(') stack.push('(')
        else {
            let sum = 0
            let top = stack.pop()
            while (top !='('){
                sum+= top
                top = stack.pop()
            }
            if (sum == 0) sum =1
            else sum *= 2
            stack.push(sum)
        }
    }
    return stack.reduce((p,v)=>p+v, 0)
};
```
