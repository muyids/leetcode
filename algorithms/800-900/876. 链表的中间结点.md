给定一个带有头结点 head 的非空单链表，返回链表的中间结点。

如果有两个中间结点，则返回第二个中间结点。

---

快慢指针

```javascript
var middleNode = function(head) {
    let slow = quick = head
    while (quick!=null){
        quick = quick.next
        if (quick == null){
            return slow
        }
        quick = quick.next
        slow = slow.next
    }
    return slow
};
```
