# 2359. 找到离给定两个节点最近的节点







# 基环树



**基环树**：每个点 只有一个出边



- N 个点 N-1 条边 的连通图 一定是一棵 **树**

- N 个点 N 条边的连通图一定是一棵**基环树**



有向基环树 从 a点 到b 点的路径是唯一的。因为这个性质，本题没有必要用 BFS 去实现，直接迭代即可。、



## 代码实现



```java
class Solution {
    public int closestMeetingNode(int[] g, int a, int b) {
        int n = g.length;
        int d1[] = new int[n];
        int d2[] = new int[n];
        Arrays.fill(d1, -1);
        Arrays.fill(d2, -1);
        d1[a] = 0;
        d2[b] = 0;
        
        while (g[a] != -1 && d1[g[a]] == -1 ) {
            d1[g[a]] = d1[a] + 1;
            a = g[a];
        }

        while (g[b] != -1&& d2[g[b]] == -1){
            d2[g[b]]= d2[b] + 1;
            b = g[b];
        }

        int d = -1;
        int minv = Integer.MAX_VALUE;
        for (int i = 0; i< n; i++){
            if (d1[i] != -1 && d2[i] != -1){
                int val = Math.max(d1[i], d2[i]);
                if (val < minv) {
                    minv = val;
                    d = i;
                }
            }
        }
        return d;
    }
}
```





# BFS



## 代码实现



```java
class Solution {

    int INF = 0x3f3f3f3f;

    int[] bfs(int[] g, int a) {
        int[] d = new int[100010];

        boolean[] st = new boolean[100010];
        Arrays.fill(d, INF);

        Deque<Integer> q = new ArrayDeque<>();
        q.addFirst(a);
        st[a] = true;
        int level = 0;
        while (!q.isEmpty()) {
            int k = q.size();
            while (k-- > 0) {
                int t = q.pollLast();
                d[t] = level;
                if (g[t] != -1 && !st[g[t]]) {
                    st[g[t]] = true;
                    q.addFirst(g[t]);
                }
            }
            level++;
        }
        return d;

    }

    public int closestMeetingNode(int[] g, int a, int b) {
        int A[] = bfs(g, a);
        int B[] = bfs(g, b);

        int minv = INF;
        int res = -1;
        for (int i = 0; i < g.length; i++) {
            if (A[i] == INF || B[i] == INF) continue;
            int val = A[i] > B[i] ? A[i] : B[i];
            if (val < minv) {
                minv = val;
                res = i;
            }
        }

        return res;
    }
}

```
