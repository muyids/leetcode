
给定一个正整数 N，试求有多少组连续正整数满足所有数字之和为 N?

示例 1:

输入: 5
输出: 2
解释: 5 = 5 = 2 + 3，共有两组连续整数([5],[2,3])求和后为 5。
示例 2:

输入: 9
输出: 3
解释: 9 = 9 = 4 + 5 = 2 + 3 + 4
示例 3:

输入: 15
输出: 4
解释: 15 = 15 = 8 + 7 = 4 + 5 + 6 = 1 + 2 + 3 + 4 + 5
说明: 1 <= N <= 10 ^ 9

### 等差数列求和公式 + 二分查找

等差数列求和公式

`sum = n * a + n * (n-1) /2`

首项 a, 求连续n项想加和为N，二分

```cpp
class Solution {
public:
    int consecutiveNumbersSum(int N) {
        int ans = 0;
        for (int a = 1; a <= N; a++) {
            int l = 1, r = N;
            if (l == r) {
                if (a == N) ans++;
                continue;
            }
            while (l < r) {
                int mid = l + (r - l) / 2;
                int sum = mid * a + mid * (mid - 1) / 2;
                if (sum > N) r = mid;
                else if (sum < N) l = mid + 1;
                else {
                    ans++;
                    break;
                }
            }

        }
        return ans;
    }
};
```

## 一眼看出本质

```cpp
class Solution {
public:
    int consecutiveNumbersSum(int N) {
        int ans = 0;
        for (int i = 1; N > 0; N-=i, i++)
            ans += (N % i == 0);
        return ans;
    }
};
```
