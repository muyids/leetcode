#### [869. 重新排序得到 2 的幂](https://leetcode.cn/problems/reordered-power-of-2/)





**DFS思路**




```java
class Solution {
    List<Integer> vt;
    boolean[] st;

    public boolean reorderedPowerOf2(int n) {
        vt = new ArrayList<>();
        while (n > 0) {
            vt.add(n % 10);
            n /= 10;
        }
        st = new boolean[vt.size()];

        return dfs(vt, st, 0, 0);
    }

    boolean dfs(List<Integer> vt, boolean[] st, int d, int x) {
        if (d == vt.size()) {
            return isTwoMe(x);
        }
        boolean f = false;
        for (int i = 0; i < st.length; i++) {
            if (st[i]) {
                continue;
            }
            if (vt.get(i) == 0 && d == 0) continue;
            st[i] = true;
            int k = vt.get(i);
            x = x * 10 + k;
            f |= dfs(vt, st, d + 1, x);
            x = x / 10;
            st[i] = false;
        }
        return f;
    }

    boolean isTwoMe(int x) {
        if (x == 1) return true;
        if ((x & 1) == 1) return false;
        return isTwoMe(x >> 1);
    }
}
```
