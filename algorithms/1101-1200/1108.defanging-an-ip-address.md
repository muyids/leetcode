

# 双指针



## 代码实现



### java

```java
class Solution {
    public String defangIPaddr(String address) {
        char[] arr = address.toCharArray();
        char[] res = new char[arr.length + 6];
        for (int i = 0, j = 0; i < arr.length; i++) {
            if (arr[i] != '.') res[j++] = arr[i];
            else {
                res[j++] = '[';
                res[j++] = arr[i];
                res[j++] = ']';
            }
        }
        return String.copyValueOf(res);
    }
}
```
