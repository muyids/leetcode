
不使用运算符 + 和 - ​​​​​​​，计算两整数 ​​​​​​​a 、b ​​​​​​​之和。

示例 1:

输入: a = 1, b = 2
输出: 3
示例 2:

输入: a = -2, b = 3
输出: 1

位运算操作

```javascript
var getSum = function (a, b) {
    while (b != 0) {
        let res = (a & b) << 1;
        a = a ^ b;
        b = res;
    }
    return a
};
```
