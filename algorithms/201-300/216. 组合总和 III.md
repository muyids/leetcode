找出所有相加之和为 n 的 k 个数的组合。组合中只允许含有 1 - 9 的正整数，并且每种组合中不存在重复的数字。

说明：

- 所有数字都是正整数。
- 解集不能包含重复的组合。 

```case
示例 1:

输入: k = 3, n = 7
输出: [[1,2,4]]
示例 2:

输入: k = 3, n = 9
输出: [[1,2,6], [1,3,5], [2,3,4]]
```

---

```cpp
class Solution {
public:

    vector<int> path;
    vector<vector<int>> ans;
    bool st[10];
    vector<vector<int>> combinationSum3(int k, int n) {
        dfs(k, n, -1);
        return ans;
    }

    void dfs(int k, int n, int last){
        if (n < 0) return;
        if (k == 0) {
            if (n == 0)  ans.push_back(path);
            return;
        }
        for (int i = 1; i<=9; i++){
            if (!st[i] && i > last){
                st[i] = true;
                path.push_back(i);
                dfs(k-1, n - i, i);
                path.pop_back();
                st[i] = false;
            }
        }
    }
};
```
