给定两个字符串 s 和 t ，编写一个函数来判断 t 是否是 s 的字母异位词。

示例 1:

输入: s = "anagram", t = "nagaram"
输出: true

用两个字典分别记录两个字符串字母出现次数，比较字典是否相等

```go
func isAnagram(s string, t string) bool {
	if len(s) != len(t) {
		return false
	}
	a1 := make([]int, 26)
	a2 := make([]int, 26)
	start := int('a')
	for i := 0; i < len(s); i++ {
		a1[int(s[i])-start] += 1
		a2[int(t[i])-start] += 1
	}
	for i := 0; i < 26; i++ {
		if a1[i] != a2[i] {
			return false
		}
	}
	return true
}
```
