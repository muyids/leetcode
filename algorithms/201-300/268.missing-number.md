给定一个包含 0, 1, 2, ..., n 中 n 个数的序列，找出 0 .. n 中没有出现在序列中的那个数。

示例 1:

输入: [3,0,1]
输出: 2

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/missing-number

---

两个相同数异或为0

```go
func missingNumber(nums []int) int {
	l := len(nums)
	ans := 0
	for i := 0; i <= l-1; i++ {
		ans = ans ^ nums[i] ^ i
	}
	ans = ans ^ l
	return ans
}
```
