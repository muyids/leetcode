
给定一个排序链表，删除所有重复的元素，使得每个元素只出现一次。

```cpp
示例 1:

输入: 1->1->2
输出: 1->2
示例 2:

输入: 1->1->2->3->3
输出: 1->2->3
```

---

值相同时，删除后一个节点即可

```cpp
class Solution {
public:
    ListNode* deleteDuplicates(ListNode* head) {
        ListNode *p = head;
        while (p && p->next){
            if (p->val == p->next->val){
                p->next= p->next->next;
            } else p=p->next;
        }
        return head;
    }
};
```
