将两个有序链表合并为一个新的有序链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。 

示例：

输入：1->2->4, 1->3->4
输出：1->1->2->3->4->4

递归遍历

```cgo
var mergeTwoLists = function(l1, l2) {
    if (!l1) return l2;
    if (!l2) return l1;
    let result = new ListNode();
    if (l1.val < l2.val) {
        result.val = l1.val
        result.next = mergeTwoLists(l1.next, l2)
    } else {
        result.val = l2.val;
        result.next = mergeTwoLists(l2.next, l1)
    }
    return result;
};
```
