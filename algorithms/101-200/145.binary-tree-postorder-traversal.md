给定一个二叉树，返回它的 后序 遍历。

示例:

输入: [1,null,2,3]  
   1
    \
     2
    /
   3 

输出: [3,2,1]
进阶: 递归算法很简单，你可以通过迭代算法完成吗？

后序遍历

左 右 根

## 递归写法

```javascript
var postorderTraversal = function(root) {
    let ans = []
    function dfs(root){
        if (!root) return
        dfs(root.left)
        dfs(root.right)
        ans.push(root.val)
    }
    dfs(root)
    return ans
};
```
