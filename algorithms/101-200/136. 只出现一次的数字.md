给定一个非空整数数组，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。

说明：

你的算法应该具有线性时间复杂度。 你可以不使用额外空间来实现吗？

```case
示例 1:

输入: [2,2,1]
输出: 1
示例 2:

输入: [4,1,2,1,2]
输出: 4
```

---

### 解题思路

异或运算

1. 任何数与0异或结果都为他本身
2. 任何两个相同的数异或结果都为0
3. 异或运算满足交换律和结合律

### 代码

cpp

```cpp
class Solution {
public:
    int singleNumber(vector<int>& nums) {
        int ans = 0;
        for (auto n : nums){
            ans ^= n;
        }
        return ans;
    }
};
```

javascript

```javascript
var singleNumber = function(nums) {
    let ans = nums[0]
    for (let i =1; i< nums.length; i++) ans ^= nums[i]
    return ans
};
```
