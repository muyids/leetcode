给定一个Excel表格中的列名称，返回其相应的列序号。

例如，

    A -> 1
    B -> 2
    C -> 3
    ...
    Z -> 26
    AA -> 27
    AB -> 28 
    ...
示例 1:
输入: "A"
输出: 1

示例 2:
输入: "AB"
输出: 28

示例 3:
输入: "ZY"
输出: 701

进制计算

```javascript
var titleToNumber = function (s) {
    let sum = 0
    for (let i = 0; i < s.length; i++) {
        sum = sum * 26 + (s[i].charCodeAt(0) - 'A'.charCodeAt(0) + 1)
    }
    return sum
};
```
