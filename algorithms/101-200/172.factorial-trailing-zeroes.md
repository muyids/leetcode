给定一个整数 n，返回 n! 结果尾数中零的数量。

示例 1:

输入: 3
输出: 0
解释: 3! = 6, 尾数中没有零。
示例 2:

输入: 5
输出: 1
解释: 5! = 120, 尾数中有 1 个零.

求末尾多少个0，也就是能拆成多少对2和5

注意 5 * 5 * ...的情况

```javascript
/**
 * @param {number} n
 * @return {number}
 */
var trailingZeroes = function(n) {
    let ans = 0
    while (n) {
        ans += Math.floor(n / 5)
        n = Math.floor(n / 5)
    }
    return ans
};
```

```cpp
class Solution {
public:
    int trailingZeroes(int n) {
        long long ans = 0;
        for (int i = 1; pow(5, i) <= n; i++){
            ans += n / pow(5, i);
        }
        return ans;
    }
};
```
